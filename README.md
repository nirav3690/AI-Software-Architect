# IdeaToMVP – AI Software Architect

An AI-powered platform that converts startup ideas into complete MVP plans.

## Features
- AI-generated feature breakdown
- Technology stack recommendations  
- Database schema generation
- Development roadmap creation
- PDF export of MVP plans
- User dashboard and plan history

## Tech Stack
- **Frontend:** React.js
- **Backend:** Python, Flask
- **Database:** PostgreSQL
- **AI:** Groq API (Llama 3.3)
